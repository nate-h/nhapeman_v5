## Recommended Size
- Aspect ratio: 2:3 (portrait)
- Minimum width: 400px
- Format: JPG or PNG

## Where to Find Book Covers
- Amazon book pages (high resolution)
- Google Images (search "book title cover high resolution")
- Publisher websites
- Goodreads

The component will display the book title as a fallback if no image is found.
